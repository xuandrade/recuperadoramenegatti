# Recuperadora Menegatti

Site institucional - engenharia de precisao, usinagem, retifica e recuperacao de pecas.

## Stack
- React 18 + Vite
- Tailwind CSS
- Framer Motion
- lucide-react

## Setup
```bash
npm install
npm run dev
```

Coloque a logo oficial em `public/logo.png` (fallback automatico para `logo.svg`).
